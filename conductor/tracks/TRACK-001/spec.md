# TRACK-001: 练功记录页 + 技能库

> **优先级**: P0 · **状态**: 🚧 in-progress · **来源**: [TODOS.md](../../../TODOS.md)

## 功能概述

新增一套**个人成长记录**功能的第一阶段：练功记录页 + 双反打法技能库。

## 用户故事

- 作为一名乒乓球爱好者，我希望记录每次练功的日期、时长、地点和陪练，以便追踪自己的训练投入
- 我希望能从预置的技能库中勾选本次练习的项目，并记录练习量和心得
- 我希望通过日历/列表回看历史练功记录
- 我希望看到累计练功次数、总时长和各技能练习频次的统计

## 验收标准

### 数据层
- [ ] 数据库新建 `skills` 表：id, category (基本功/技战术), name, style (打法类型，默认"双反"), sort_order
- [ ] 数据库新建 `training_logs` 表：id, player_id, date, duration_minutes, location, partner, energy_rating (1-5), feel_rating (1-5), notes, created_at
- [ ] 数据库新建 `training_log_skills` 表（多对多）：training_log_id, skill_id, practice_amount (练习量), notes (该项心得)
- [ ] 预置 ~30 项双反打法技能数据写入 skills 表

### 后端 API
- [ ] `GET /api/skills` — 获取技能库列表（按 category + sort_order 排序）
- [ ] `POST /api/training-logs` — 创建练功记录（含关联技能）
- [ ] `GET /api/training-logs` — 获取当前用户的练功记录列表（支持分页/筛选）
- [ ] `GET /api/training-logs/:id` — 获取单条练功记录详情
- [ ] `PUT /api/training-logs/:id` — 更新练功记录
- [ ] `DELETE /api/training-logs/:id` — 删除练功记录
- [ ] `GET /api/training-stats` — 获取统计数据（次数、总时长、技能频次）

### 前端页面
