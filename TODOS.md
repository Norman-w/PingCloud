## 🚧 P0 · 练功记录页（当前优先）

记录自己每一次练功的信息，配套一份「双反打法」的技能库。

### 页面功能
- [ ] 新增一次练功记录：日期、时长、地点、陪练/教练、主观状态（体力/手感 1-5 星）
- [ ] 每次练功勾选本次练的**基本功 / 技战术**项（从下方技能库选）
- [ ] 每项可写练习量（组数/球数/分钟）+ 简短心得
- [ ] 练功日历 / 列表视图，可回看历史
- [ ] 统计：累计练功次数、总时长、各技能练习频次

### 技能库（双反 = 横板双面反胶打法）—— 待「上架」

#### 一、基本功
- [ ] 正手攻球（近台快攻）
- [ ] 反手拨球 / 快拨
- [ ] 正手拉球（前冲弧圈）
- [ ] 正手拉球（加转弧圈）
- [ ] 反手拉球 / 反手弧圈
- [ ] 正手搓球
- [ ] 反手搓球
- [ ] 摆短
- [ ] 劈长
- [ ] 台内挑打
- [ ] 反手拧拉（香蕉球）
- [ ] 正手挡球 / 快带
- [ ] 反手弹击
- [ ] 正手扣杀
- [ ] 发球：正手发球
- [ ] 发球：反手发球
- [ ] 发球：逆旋转发球
- [ ] 接发球（搓/挑/拧的判断与选择）
- [ ] 步法：单步 / 并步 / 跳步 / 交叉步 / 垫步

#### 二、技战术
- [ ] 发球抢攻（正手 / 侧身）
- [ ] 接发球抢攻（拧拉上手 / 挑打）
- [ ] 反手相持对拉
- [ ] 反手相持中转正手进攻
- [ ] 侧身抢攻（反手位侧身正手爆冲）
- [ ] 摆短控制 + 抢先上手
- [ ] 劈长压反手底线
- [ ] 落点变化：追身 / 大角度 / 中路结合部
- [ ] 节奏与旋转变化（快慢结合、加转与不转）
- [ ] 相持中的转换与反拉防守

> 💡 「上架」= 把以上技能作为预置数据写入数据库（建议新建 `skills` 表，含 `category` 基本功/技战术、`name`、`style` 双反），供练功记录页勾选。
